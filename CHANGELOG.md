# Changelog - Depizza Town Menu

## Latest Update - Product Modal Improvements

### ✅ Nutrition Facts with Progress Bars
- **NEW**: Added visual progress bars for all nutrition values
- Calories: Orange to red gradient bar (max 1000 cal)
- Protein: Blue bar (max 50g)
- Carbs: Green bar (max 100g)
- Fat: Yellow bar (max 50g)
- Smooth animation when bars appear (500ms transition)
- Responsive design - works on all screen sizes

### ✅ Compact Modal Design
- Reduced image height: 72px → 56px (224px total)
- Smaller font sizes throughout:
  - Title: 4xl → 3xl on desktop, 3xl → 2xl on mobile
  - Price: 4xl → 3xl on desktop, 3xl → 2xl on mobile
  - Description: lg → base on desktop, base → sm on mobile
  - Section headings: 2xl → lg
  - Badges: sm → xs
- Reduced padding: p-8 → p-6 on desktop, p-6 → p-5 on mobile
- Reduced spacing between sections: mt-8 → mt-5/mt-6
- Badge padding: py-2 px-4 → py-1.5 px-3

### ✅ Better Typography
- Changed font from Geist to **Inter** (Google Font)
- More readable and professional
- Better spacing and weight distribution
- Optimized for both light and dark themes

### ✅ Scrollbar Improvements
- Removed all visible scrollbars from modal

- Content scrolls smoothly without visual scrollbars
- Clean, modern appearance

### ✅ Responsive Design
- All progress bars adapt to screen size

- Padding and spacing adjust for mobile
- Modal height: max-h-[85vh] for better mobile viewing
- Touch-friendly interface
